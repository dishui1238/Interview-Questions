### 4.双指针技巧

经典问题：反转数组中的元素。比如数组为 ['l', 'e', 'e', 't', 'c', 'o', 'd', 'e']，反转之后变为 ['e', 'd', 'o', 'c', 't', 'e', 'e', 'l']。

思考：使用双指针技巧，其思想是分别将两个指针分别指向数组的开头及末尾，然后将其指向的元素进行交换，再将指针向中间移动一步，继续交换，直到这两个指针相遇。

```js
function reverseString(s) {
  let i = 0,
    j = s.length - 1;
  while (i < j) {
    // const temp = s[i];
    // s[i] = s[j];
    // s[j] = temp;
    [s[i], s[j]] = [s[j], s[i]]; // 不使用中间变量
    i += 1;
    j -= 1;
  }

  return s;
}
```
